What is Ajax?
Ajax (Asynchronous JavaScript and XML or JSON ) is a technique for creating interactive web applications by exchanging data with a server in the background,
without requiring a full page reload. It is a combination of several technologies, including JavaScript, HTML,
CSS, and the XMLHttpRequest object. Ajax allows you to update parts of a web page without reloading
the entire page, making it a powerful tool for creating dynamic and interactive web applications.

Https verbs
1. GET: The GET method is used to request data from a server. It is typically used to retrieve data from a server and is the
most common HTTP method used in web applications.
2. POST: The POST method is used to send data to a server to create a new resource. It is typically used to create a new
resource on the server, such as creating a new user account or posting a new comment.
3. Delete: The DELETE method is used to delete a resource on a server. It is typically used to delete a resource that  
has already been created, such as deleting a user account or removing a comment.
